Medisage Website

One of the most interesting projects I've worked on was doctor-patient interaction platform. The frontend UI was built using React and Next. This project was particularly exciting because it involved creating a user-friendly interface for managing patient cases, live webinars led by experts, reading medical journals, subscribing to newsletters, practice pages, wallet system, all within a single application. The objective was to develop a responsive platform facilitating seamless communication between doctors and patients. The positive feedback from doctors highlighted the success of the project and its value in enhancing healthcare delivery. Overall, this project demonstrated the power of React in building complex, responsive, and user-friendly web applications, and it remains one of the most rewarding experience in my career as a frontend developer.

Medisage Admin Panel

One of the most significant projects I've worked on was developing the Medisage admin panel which was mainly built for internal medisage users and maintain administrative tasks for the Medisage platform. The admin panel was designed to be a comprehensive CRUD (Create, Read, Update, Delete) system with a structured UI. For displaying backend data in tabular form, I used Bootstrap tables, and for styling, I used CSS. One of the main highlights of this project was implementing role-based access control which ensured that different medisage user can only access the specific modules they were authorized for, providing secure and efficient data handling. Working on the Medisage admin panel was a thrilling experience as it allowed me to delve deep into Vue and build a system that greatly facilitated the management of the Medisage platform.
